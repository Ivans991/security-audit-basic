# Tools Used in the Security Audit

This document describes the tools selected for the execution of the basic security audit and the purpose of each one.

## Nmap
- Purpose: Network scanning and identification of open ports and services.
- Justification: Nmap is a widely used and reliable tool for network reconnaissance and service discovery.

## Netstat
- Purpose: Review of active network connections and listening ports on the local system.
- Justification: Netstat allows visibility into current network activity and helps identify unexpected or suspicious connections.

## Operating System Firewall
- Purpose: Verification of firewall status and basic rule configuration.
- Justification: The firewall is a primary defense mechanism that controls incoming and outgoing traffic.

## Antivirus / Windows Defender
- Purpose: Verification of active antivirus protection and basic security status.
- Justification: Antivirus software helps detect and prevent known malicious threats.

## Built-in System Tools
- Purpose: Review of user accounts, permissions, and system configurations.
- Justification: Native system tools provide reliable information without introducing external risks.
