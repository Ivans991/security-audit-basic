## Objective
The objective of this project is to perform a basic, non-intrusive security audit on a local system in order to identify potential security weaknesses related to open ports, running services, user accounts, and basic protection configurations.

## System Under Audit
- System type: Local workstation
- Operating system: Windows 11
- Audit environment: Personal system
- Audit type: Educational and non-production

## Included Activities
- Identification of open ports and active services
- Review of current network connections
- Basic analysis of user accounts and permissions
- Review of firewall and antivirus status
- Identification of potential security risks
- Documentation of findings and recommendations

## Excluded Activities
- Penetration testing or exploitation
- Denial of Service attacks
- Password cracking
- Malware development or deployment
- Social engineering attacks

## Ethical Statement
This audit is conducted strictly for educational purposes on a system owned by the author. No unauthorized access, exploitation, or malicious activity is performed.
