# Netstat Analysis

## Command Executed
netstat -ano

## Purpose
The purpose of this command is to identify active network connections, listening ports, and associated processes on the local system.

## General Observations
- Multiple TCP connections were observed in LISTENING and ESTABLISHED states.
- Several UDP entries were present without connection states.
- Local addresses included common ports such as 80, 443, and dynamic ports.

## Notes
No intrusive actions were taken. This analysis is limited to observation and documentation of network activity.
